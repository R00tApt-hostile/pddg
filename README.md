# Decentralized & Privacy Tools Directory

A static website listing decentralized apps, open-source privacy tools, and de-Googled alternatives. Built with old Reddit aesthetics and designed to run on GitHub Pages.

## Features

- **Categorized Listings**: Browse tools by category (Messaging, Browsers, Search Engines, etc.)
- **Search Functionality**: Find specific tools by name or description
- **Sorting Options**: Sort by name, category, or privacy level
- **Statistics Display**: See counts for total tools, open-source, and decentralized options
- **Responsive Design**: Works on desktop and mobile devices
- **Old Reddit Aesthetic**: Minimalist, text-focused interface
